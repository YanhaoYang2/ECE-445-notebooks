# Team 9 Ant-weight 3D Printed Battlebot

A sub-2 lb antweight combat robot featuring a servo-driven lifting wedge, ESP32-S3 BLE wireless control, dual H-bridge motor drivers, and a fully 3D-printed PLA chassis: built for ECE 445 Senior Design at UIUC. The custom PCB integrates power regulation, drivetrain control, and weapon actuation into a compact, competition-ready platform.
